# Funciones incorporadas

Cada una de las siguientes palabras es una función incorporada de Python, no recibirá un error si utiliza alguna de las siguientes funciones incorporadas como nombre de una variable pero si anulará el comportamiento de esa función. Para obtener una lista de las funciones incorporadas, para esto debe importar el módulo `builtins` y luego utilizar la función `dir()` para obtener la lista del nombre de las funciones incorporadas en el módulo.
