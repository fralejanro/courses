# Programación Orientada a Objetos (POO)

La programación orientada a objetos construye sus programas alrededor de sus datos, es decir, alrededor de los objetos y su comportameniento.

## Abstracción 

Un concepto esencial de la programación orientada a objetos  es la abstracción. La abstracción se refiere a la forma en que puede construir objetos que describen un comportamiento unico y que tienen la capacidad de comunicarse e interactuar con otros objetos.

## Principios de la Programación Orientada a Objetos

La programación orientada a objetos se basa en tres principios que le ayudarán a implementar el modelo orientado a objetos. Estos principios son encapsulación, herencia y polimorfismo.

### Encapsulación

La encapsulación es la forma que se utiliza para proteger el código y los datos que manipula.

### Herencia

La herencia es el proceso por el cual un objeto adquiere las propiedades de otro objeto

### Polimorfismo

El polimorfismo es una característica que permite implementar una interfaz para definir una clase de acciones generales.
